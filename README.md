# Codex History

English | [简体中文](README.zh-CN.md)

A Codex skill for recovering missing conversations, finding exact earlier wording, exporting clean transcripts, and diagnosing continuity problems after context compaction.

## Install

```bash
npx skills add YowaiCoder/codex-history --global --agent codex
```

## Use with Codex

Ask Codex naturally, for example:

- “Recover my missing conversations.”
- “Export this conversation as a clean transcript.”
- “What exactly did I say earlier about `<topic>`?”
- “What exactly did you say before the context was compacted?”
- “Why did you repeat yourself after compaction, and how can we prevent it?”
