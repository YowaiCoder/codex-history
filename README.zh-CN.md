# Codex History

[English](README.md) | 简体中文

一个 Codex 技能，用于找回消失的对话、查找之前的原话、导出干净的对话记录，以及诊断上下文压缩后的连续性问题。

## 安装

```bash
npx skills add YowaiCoder/codex-history --global --agent codex
```

## 在 Codex 中使用

直接用自然语言向 Codex 提问，例如：

- “帮我找回消失的对话。”
- “将当前对话导出为干净的对话记录。”
- “我之前关于 `<主题>` 的原话是什么？”
- “上下文压缩前你到底是怎么说的？”
- “为什么你在上下文压缩后重复回复？怎么避免？”
